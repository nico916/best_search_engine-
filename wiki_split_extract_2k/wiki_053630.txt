La loi stable ou loi de Lévy tronquée , nommée d' après le mathématicien Paul Lévy , est une loi de probabilité utilisée en mathématiques , physique et analyse quantitative ( finance de marché ) .
On dit qu' une variable aléatoire réelle X { \ displaystyle X } est de loi stable si elle vérifie l' une des 3 propriétés équivalentes suivantes : Pour tous réels strictement positifs A { \ displaystyle A } et B { \ displaystyle B } , il existe un réel strictement positif C { \ displaystyle C } et un réel D { \ displaystyle D } tels que les variables aléatoires A X 1 + B X 2 { \ displaystyle AX_ { 1 } + BX_ { 2 } } et C X + D { \ displaystyle CX + D } aient la même loi , où X 1 { \ displaystyle X_ { 1 } } et X 2 { \ displaystyle X_ { 2 } } sont des copies indépendantes de X { \ displaystyle X } .
